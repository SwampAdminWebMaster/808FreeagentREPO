# 808 MAFIA . LIVE

**Underground Bass Culture | Collective | Community**

---

## Overview

808 MAFIA . LIVE is a digital hub for the underground bass, music, and art collective. This site serves as the primary gateway for:

- 🎵 Music & Production
- 🎪 Events & Experiences  
- 🎬 Media & Content
- 🌐 Community Connection

---

## Features

✨ **Cyberpunk Aesthetic** - Premium hacker/mafia visual theme with glitch effects  
⚡ **Terminal Boot Screen** - Immersive OS-style loading sequence  
🎨 **Responsive Design** - Mobile-friendly layout  
🔗 **Social Integration** - Direct links to Instagram, SoundCloud, YouTube  
📧 **Contact Form** - Direct communication channel

## Customization

### Colors
- **Primary Green**: `#0f0` (neon green)
- **Accent Magenta**: `#ff006e` (hot pink)
- **Accent Cyan**: `#00d9ff` (electric blue)
- **Background**: `#000` (pure black)

### Links
Update social links in `index.html`:
```html
<a href="https://instagram.com/YOUR_HANDLE" target="_blank">Instagram</a>
<a href="https://soundcloud.com/YOUR_HANDLE" target="_blank">SoundCloud</a>
<a href="https://youtube.com/YOUR_CHANNEL" target="_blank">YouTube</a>
<a href="mailto:YOUR_EMAIL@gmail.com">Contact</a>
```

## Development

### Local Testing
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js http-server
npx http-server
```

Then visit: `http://localhost:8000`
